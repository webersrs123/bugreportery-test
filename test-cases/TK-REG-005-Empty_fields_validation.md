## ТК-REG-005: Регистрация с пустыми полями обязательными полями

**Предусловие**: Пользователь не авторизован

**Окружение**: Windows 11, Chrome 146 (режим инкогнито), 1920x1080

**Дата:** 22.03.2026

**Шаги:**
1. Открыть страницу https://bugreportery.qatest.tech
2. В хедере нажать кнопку "Регистрация"
3. Поле Имя пользователя оставить пустым
4. Поле Email оставить пустым
5. Поле Пароль оставить пустым
6. Нажать кнопку "Зарегистрироваться"

  **Ожидаемый результат:**
- Регистрация не выполнена
- Отображается сообщение об ошибке: "Введите email" / "Введите пароль" / "Заполните все обязательные поля"
- Поля "Email" и "Пароль" подсвечено красным
- фокус на поле "Email"
