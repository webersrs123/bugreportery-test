
## ТК-CREATE-007: Попытка сохранить баг-репорт без Ожидаемого результата

**Предусловие:** Пользователь авторизован c email test@qatest.tech

**Окружение:** Windows 11, Chrome 146 (режим инкогнито), 1920x1080

**Дата:** 24.03.2026

**Шаги:**
1. Открыть страницу https://bugreportery.qatest.tech
2. Поле "Ожидаемый результат" оставить бустым
3. Заполнить остальные обязательные поля
4. Нажать кнопку "Сохранить репорт" 

**Ожидаемый результат:**
- Баг-репорт не сохранён
- Появляется сообщение об ошибке для поля "Ожидаемый результат"
- Поле "Ожидаемый результат" подсвечивается.
