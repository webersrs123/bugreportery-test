
## ТК-CREATE-001: Создание баг-репорта с заполнением всех обязательных полей

**Предусловие:** Пользователь не авторизован

**Окружение:** Windows 11, Chrome 146 (режим инкогнито), 1920x1080

**Дата:** 22.03.2026

**Шаги:**
1. Открыть страницу https://bugreportery.qatest.tech
2. Заполнить все обязательные поля
3. Нажать кнопку "Сохранить репорт" 

**Ожидаемый результат:**
- Баг-репорт сохранён
- Появляется сообщение об успешном создании
- В списке багов отображается новая запись с уникальным ID (например, BUG-001)
- Все введённые данные соответствуют сохранённым
