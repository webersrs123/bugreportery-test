# Баг-репорт BUG-009

## Название
UI/UX / Отсутствует функция восстановления пароля на форме входа

| Проект | BugReportery |
|--------|----------------|
| Серьёзность | Minor |
| Приоритет | Medium |
| Тип бага | UI/UX / Функциональность |
| Статус | Открыт |

## Описание
На форме входа отсутствует ссылка или кнопка «Забыли пароль?». Пользователь не может восстановить доступ к учётной записи, если забыл пароль.

## Окружение
- **Устройство:** Desktop
- **ОС:** Windows 11 Pro
- **Браузер:** Google Chrome 146.0.7680.80
- **URL:** https://bugreportery.qatest.tech

## Шаги воспроизведения
1. Открыть страницу https://bugreportery.qatest.tech
2. Нажать кнопку «Вход».
3. Обратить внимание на содержимое модального окна входа.

## Фактический результат
В модальном окне присутствуют только поля «Email», «Пароль» и кнопки «Войти» и «Закрыть». Отсутствует ссылка на восстановление пароля.

## Ожидаемый результат
Под формой входа должна быть ссылка «Забыли пароль?» или аналогичная, ведущая к форме восстановления (например, отправка инструкции на email).
