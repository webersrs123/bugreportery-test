# Баг-репорт BUG-004

## Название
UI / Модальные окна «Вход» и «Регистрация» не закрываются при клике вне области

| Проект | BugReportery |
|--------|----------------|
| Серьёзность | Minor |
| Приоритет | Medium |
| Тип бага |  UI/UX|
| Статус | Открыт |

## Описание
При открытии модального окна (вход или регистрация) щелчок мышью за пределами окна не приводит к его закрытию. Ожидается, что клик по фону (overlay) должен закрывать модалку.

## Окружение
- **Устройство:** Desktop
- **ОС:** Windows 11 Pro
- **Браузер:** Яндекс Браузер 25.12.4.1216 (64‑бит) / Google Chrome 146.0.7680.80 (64‑бит) Режим инкогнито
- **URL:** https://bugreportery.qatest.tech

## Шаги воспроизведения
1. Открыть страницу https://bugreportery.qatest.tech
2. В хедере нажать кнопку "Регистрация"/ "Вход"
3. После открытья модального окна "регистрация" кликнуть мышью в за пределами окна "регистрация"
4. Повторить шаги 1-3 для кнопки "Вход"

## Фактический результат
- Модальные окна "Регистрация" / "Вход" остаются открытыми
 
## Ожидаемый результат ##
- Для удобства пользователя ожидается закрытие окон "Регистрация" и "Вход" после клика в не области окна.
