# Баг-репорт BUG-005

## Название
UI / Модальное окно регистрации — кнопки «Регистрация» и «Вход» визуально сливаются 

| Проект | BugReportery |
|--------|----------------|
| Серьёзность | Minor |
| Приоритет | Medium |
| Тип бага | UI/UX |
| Статус | Открыт |

## Описание
В модальном окне регистрации кнопки «Регистрация» и «Вход» расположены слишком близко друг к другу, из‑за чего они визуально сливаются. Это затрудняет взаимодействие и ухудшает восприятие интерфейса.

## Окружение
- **Устройство:** Desktop
- **ОС:** Windows 11 Pro
- **Браузер:** Google Chrome 146.0.7680.80, Яндекс Браузер 25.12.4.1216
- **URL:** https://bugreportery.qatest.tech

## Шаги воспроизведения
1. Открыть страницу https://bugreportery.qatest.tech
2. В хедере нажать кнопку "Регистрация"
3. Обратить внимание на область, где расположены кнопки «Регистрация» и «Вход» (низ модального окна).

## Фактический результат
Кнопки расположены вплотную друг к другу, сливаются, не имеют достаточного отступа.

## Ожидаемый результат
Кнопки должны иметь чёткие визуальные границы и достаточный отступ, чтобы пользователь мог легко их различить и нажать.
