# Баг-репорт BUG-REG-007

## Название
Локализация: сообщение о неверном пароле отображается на английском, ожидается русский

| Проект | BugReportery |
|--------|----------------|
| Серьёзность | Minor |
| Приоритет | Medium |
| Тип бага | UI/Локализация |
| Статус | Открыт |

## Описание
При вводе неверного пароля в форме авторизации пользователь видит сообщение об ошибке на английском языке. "Invalid login credentials" Для русскоязычных пользователей сообщение должно отображаться на русском.

## Окружение
- **Устройство:** Desktop
- **ОС:** Windows 11 Pro
- **Браузер:** Яндекс Браузер 25.12.4.1216 (64‑бит) / Google Chrome 146.0.7680.80 (64‑бит) Режим инкогнито
- **URL:** https://bugreportery.qatest.tech

## Шаги воспроизведения
1. Открыть страницу https://bugreportery.qatest.tech
2. В хедере нажать кнопку "Вход"
3. В поле Email ввести: test@qatest.tech
4. В поле Пароль ввести: wrongpass123
5. Нажать кнопку "Войти"

## Фактический результат
Отображается сообщение: «Invalid login credentials» 

## Ожидаемый результат
Сообщение об ошибке должно быть на русском языке (например, «Неверный логин или пароль»).
