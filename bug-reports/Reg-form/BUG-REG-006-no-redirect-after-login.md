## Название
UI / Авторизация. Нет редиректа на страницу ЛК.

| Проект | BugReportery |
|--------|----------------|
| Серьёзность | Blocker |
| Приоритет | High |
| Тип бага | Регистрация / UI|

## Описание
После успешной авторизации нет редиректа на страницу личного кабинета, пользователь остаётся на главной странице. Кнопка "Выход" Отображается, работает. 

## Окружение
- **Устройство:** Desktop
- **ОС:** Windows 11 Pro
- **Браузер:** Яндекс Браузер 25.12.4.1216 (64‑бит) / Google Chrome 146.0.7680.80 (64‑бит) Режим инкогнито
- **URL:** https://bugreportery.qatest.tech

## Шаги воспроизведения
1. Открыть страницу https://bugreportery.qatest.tech
2. В хедере нажать кнопку "Вход"
3. В поле Имя пользователя ввести: TestUser
4. В поле Email ввести: test@qatest.tech
5. В поле Пароль ввести: ****
6. Нажать кнопку "Вход"

## Фактический результат
- Авторизация успешна
- Редирект на страницу не происходит
- Имя пользователя TestUser не отображается в хедере

## Ожидаемый результат ##
- Авторизация успешна
